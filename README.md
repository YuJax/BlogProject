# BlogProject

## Front End

- Vue 3
- TypeScript

## Back End

- Spring MVC
- MySQL

## How to run

### Front End

Run the following commands in the terminal:

```bash
cd Blog_frontEnd
npm install
cat VITE_API_URL=http://localhost:8080 > .env.local # change to your own back end url
npm run serve
```

### Back End

```bash
cd Blog
mvn spring-boot:run
```
